org-mode的安装

Emacs23之后已经默认包含有org-mode了，但是默认包含的版本往往比较低，因此建议安装一个新的org-mode。在emacs中，安装插件主要有el-get和 ELPA两种，现在将这两种方法介绍如下。

ELPA

ELPA在在emacs24及以上版本已经默认集成，ELPA默认只有gnu官方源，有些插件可能在官方源里面找不到，一般情况建议添加marmalade、melpa这两个源。

然后通过M-x list-packages会列出所有的包，然后选择org就可以安装。

el-get

el-get是一个第三方的包管理器，个人感觉el-get使用比较方便些。el-get的具体安装方法请参考github上的文档，在Windows下安装el-get会出现ssl错误，解决方法情Installation on Windows

The lisp snippets for installing el-get download the full el-get installer via https, eg the “lazy installer”:

>

but Emacs doesn’t support https out of the box on Windows. As mentioned here, the needed dlls can be retrieved from http://sourceforge.net/projects/ezwinports/files/.

Extract the dlls in bin from gnutls-3.0.9-w32-bin.zip into Emacs’ bin/

but not the .exe files, otherwise you might hit emacs bug 15905.

You will need to restart Emacs so it can notice the new dlls.

el-get安装完成后，即可使用 M-x el-get-install来安装org-mode了。
